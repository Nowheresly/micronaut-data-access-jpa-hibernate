Another option is to use https://docs.oracle.com/javaee/7/api/javax/transaction/Transactional.html[Java EE 7 javax.transaction.Transactional annotation] to mark your transaction demarcations.

But to use it, you have to include the `micronaut-data-processor` dependency in your annotation processor configuration:

[source, groovy]
.build.gradle
----
...
..
dependencies {
...
..
.
include::{sourceDir}/../javaxtransactional/build.gradle[tag="micronaut-data"]
}
----
