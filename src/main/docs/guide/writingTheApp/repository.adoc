<1> Use `javax.inject.Singleton` to designate a class a a singleton.
<2> Inject easily an `EntityManager`.
<3> All database access needs to be wrapped inside a transaction, we use Micronaut's `@TransactionalAdvice`. As the method
only reads data from the database, configure it as `readOnly = true`.
<4> This method modifies the database, define `@TransactionalAdvice` without setting `readOnly`, which, by default is `false`.
